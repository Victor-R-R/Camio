Flux principaux
📝 Création d'une liste
1. Formulaire → Server Action createList()
↓
2. Prisma insert → shareToken uuid généré
↓
3. Status → GENERATED · redirect vers aperçu
↓
4. Aperçu 3 bons → bouton Imprimer (window.print)
📱 Fiche magasinier (lien public)
Route publique /check/[token]
↓
Lookup token → affiche items (Server Component)
↓
Coche item → Server Action toggleItem()
↓
Revalidate path → mise à jour UI instantanée
⚙️ Upload logo entreprise
File input → Route Handler POST /api/upload
↓
Vercel Blob → URL stockée dans CompanySettings
